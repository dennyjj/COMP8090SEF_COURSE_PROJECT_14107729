# Task 2: Self-Study on a New Data Structure and Algorithm

## Data Structure: Matrix

A matrix is a two-dimensional data structure organized in rows and columns. This study covers the abstract data type (ADT), key operations (addition, multiplication, transpose, scalar operations), and real-world applications such as image processing, graph representation, and solving systems of equations.

## Algorithm: Timsort

Timsort is a hybrid sorting algorithm combining merge sort and insertion sort, created by Tim Peters in 2002. It is the default sorting algorithm used by Python and Java. This study covers the core concepts (runs, minrun, galloping mode, merge strategy), time complexity analysis, and step-by-step examples.
